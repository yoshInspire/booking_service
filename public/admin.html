Инструкция для администратора

  • Для изменения статуса записи используйте выпадающий список в колонке "Статус"
  • Для экспорта всех записей в Excel нажмите кнопку "Экспорт в Excel" вверху
  • Для экспорта только отфильтрованных записей нажмите кнопку "Экспорт отфильтрованных"
  • Консультации родителей проводятся только по субботам с 11:30 до 13:00
  • Длительность консультации: 20 минут на одного родителя
  • Новые записи отмечены желтым цветом
  • Подтвержденные записи отмечены зеленым цветом
  • Отмененные записи отмечены красным цветом
  • Завершенные записи отмечены синим цветом
  • Для удаления записи используйте кнопку с иконкой корзины
  • Все изменения сохраняются автоматически
